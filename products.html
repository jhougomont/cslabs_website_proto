DeltaX Benefits

Open and Modular

DeltaX is an open and modular software framework that provides an easy-to-use configurable pipeline for processing terabytes of data for AI test and training. Pipeline modules run concurrently (in parallel) and can easily be swapped out for other similar modules, so different processing algorithms or external tools can be used and compared

Fully Configurable

Supports integration of different tools to avoid vendor lock-in. And with the GUI defined as module, DeltaX allows for easy swapping of the front-end for different hardware systems and use cases

Fully Traceable and Secure

Check point and restart capabilities to avoid losing progress in case of hardware failures, power outages, etc. Full traceability of the processing chain for tracking, evaluation, and archival of results..
